//Projet 7 Openclassrooms

executez la commande "npm install" depuis les dossiers /back et /front
Ajoutez vos identifiants et nom de base de donnée a utiliser au fichier /back/.env

lancer le serveur du dossier /back avec la commande nodemon
lancer l'application du dossier /front avec la commande npm start

par defaut  les nouveaux utilisateurs sont assignés le rang 'user'
creez un utilisateur admin manuelement via l'API en ajoutant 'rank': 'admin' au corp de la requête
('http://localhost:3000/auth/signup')
